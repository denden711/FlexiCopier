### FlexiCopier README

---

**FlexiCopier**は、元のファイルの拡張子を保持しながら、単一のソースファイルを複数の異なるファイル名でコピーすることができるファイルコピーユーティリティです。簡単にファイルとディレクトリを選択できるグラフィカルユーザーインターフェース（GUI）を備えています。

#### 特徴
- **簡単なインタラクションのためのGUI**: ファイルと目的地の選択が簡単なグラフィカルインターフェース。
- **複数のコピー**: ソースファイルをテキストファイルにリストされた複数の新しいファイル名でコピーします。
- **拡張子の保持**: 新しいファイル名に元のファイルの拡張子を自動的に追加します。

#### システム要件
- **オペレーティングシステム**: Windows 7以降、MacOS、またはLinux
- **Pythonバージョン**: Python 3.6以上
- **依存関係**: Pythonの`Tkinter`モジュール（GUI操作用）、`shutil`、`os`モジュール（ファイル操作用）

#### インストール方法

1. **Pythonのインストール**: システムに Python 3.6 以上がインストールされていることを確認してください。[python.org](https://www.python.org/downloads/) からダウンロードできます。
2. **ソースコードのダウンロード**: プロジェクトページからリポジトリをクローンするか、zipファイルをダウンロードします。
   ```bash
   git clone https://github.com/yourusername/FlexiCopier.git
   cd FlexiCopier
   ```
3. **依存関係のインストール**: 
   - FlexiCopierはPython標準ライブラリ以外の外部ライブラリは必要ありません。
